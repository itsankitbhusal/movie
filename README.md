# Movie App (Next.js)

A responsive movie app built with Next.js and TypeScript.

## Packages Used

- Next.js 15, React 19, TypeScript
- Tailwind CSS
- React Query
- React paginate
- React responsive
- useHooks (for favorites)

## To Start

```bash

nvm use
pnpm install
pnpm dev
```

## Routes

- `/` - Home
- `/movies/:slug` - Movie details
